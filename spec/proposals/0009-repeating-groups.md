# GSP-0009 — moved

`0009-repeating-groups.md` was this proposal's pre-publication working title.
It was renamed to
[`0009-composite-repeating-values.md`](./0009-composite-repeating-values.md)
before the proposal was drafted out; that file is the canonical, current text.

This stub exists only so that outside links using the old filename resolve
instead of 404ing. Three published, immutable `VERIFICATION.md` records
(`registry/us/uscis/permanent-resident-card-replacement-i90/1.0.0/`,
`registry/us/uscis/travel-document-i131/1.0.0/`, and
`registry/us/uscis/adjustment-of-status-i485/1.0.0/`) reference GSP-0009 under
this pre-rename path; per [VERSIONING.md](../../VERSIONING.md) §3, a published
version directory is never edited, so the records themselves keep the old
path and this file fills it in rather than the other way around
(see GOV-1095). This stub carries no normative content of its own — see
[GSP-0009](./0009-composite-repeating-values.md) for the actual proposal.
